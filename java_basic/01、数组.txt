堆内存的特点：
1、每一个new 出来的东西都有地址
2、每个变量都有默认值
	byte,short,int long 0
	float double 0.0
	char '\u0000'
	boolean false
	引用类型 null
3、使用完毕就编程垃圾，并没有立即回收，会在垃圾回收器空闲时回收

数据操作的两个常见小问题：
      ArrayIndexOutOfBoundsException:数组索引越界异常，访问量不存在的索引
      BullPointerException:空指针异常，数组已经不再指向对内存了。
